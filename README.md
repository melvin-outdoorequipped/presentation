# NotebookLM Intelligence Brief - Separated Files

## Files Created

1. **index.html** - Main HTML structure with external CSS/JS links
2. **styles.css** - Complete stylesheet (all CSS)
3. **data.js** - Speaker script data and constants
4. **slides-content.js** - All 9 slide HTML content
5. **particles.js** - Background particle animation
6. **charts.js** - Adoption curve and live counters
7. **navigation.js** - Slide navigation logic
8. **day-toggle.js** - Before/After day view toggle
9. **features.js** - Feature cards and modal
10. **simulation.js** - Demo simulation and AI Q&A
11. **command.js** - Command palette (⌘K)

## How to Use

1. Open `index.html` in a web browser
2. All files must be in the same directory
3. No build process or server required

## Navigation

- **Arrow Keys (← →)**: Navigate between slides
- **⌘K / Ctrl+K**: Open command palette
- **S**: Toggle speaker script panel

## File Sizes

- Each file is now 2-40 KB (vs 88 KB monolithic file)
- Easier to maintain and understand
- Better version control
- Faster development workflow
